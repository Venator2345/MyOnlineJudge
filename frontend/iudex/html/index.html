Observações e exclarecimento de dúvidas


1 - E/S no Javascript

Para leitura e escrita, use os comandos deste juiz online: input e print. Eles funcionam de maneira parecida com os comandos de E/S da linguagem Python. Exemplo:

let num = input();

A variável receberá uma string, não um número. Faça a conversão se necessário.

2 - E/S no Assembly 6502

Considere um hardware com E/S mapeada em memória, como o console de viedeogame NES. A escrita é feita gravando um valor no endereço 0x2001 e a leitura da entrada ocorre ao ler do endereço 0x2000.

Cada operação de leitura pega um número por vez assim como cada operação de escrita envia um número por vez ao dispositvo de saída, que o converterá para um caractere de acordo com a tabela ASCII.

ldx #$41

stx $2001

As instruções acima vão adicionar o caractere "A" para a saída do programa.

Qualquer gravação em um endereço acima de 0x2001 será entendida como uma violação de segurança e será impedida.

3 - Mais sobre o Assembly 6502

Considere que o código escrito será executado em uma interrupção. Dessa forma, procure usar a instrução rti na última linha de código para encerrar o programa.

O processador simulado pelo Juiz é um Ricoh 2A03, não um MOS 6502. Dessa forma, não se trabalha com o modo decimal.
